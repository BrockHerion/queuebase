# Welcome to Queuebase

Queuebase is the easiest way to add background jobs to your Next.js application. It does all the hard parts of managing a job queue for you, so you can focus on building your app. [Get started](/getting-started/appdir)

- Background jobs run on YOUR server
- Supports both client and server components
- Simple, fully type-safe API for a great developer experience
- Dashboard to monitor and manage your jobs

Got questions? Join us on our [Discord](https://discord.gg/queuebase) server!

> Queuebase is in alpha. Stuff will probably break. Please report any issues you find!
